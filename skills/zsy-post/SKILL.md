# 一站式中实仪发布文案

根据已定稿的脚本内容，按「一站式中实仪」账号的四种已验证文案风格，生成适合抖音/视频号发布的短文案。

本 skill 是**内容发布阶段**的工具，前置审稿阶段用 `zsy-review`。

---

## 四种文案风格

### 类型1：痛点反问型（最常用，出镜率最高）

**公式**：政策/行业事件 + 时间节点制造紧迫感 + 反问直击客户 + 标签收尾

**模板**：
```
#话题 新标/新规/新技术 + 时间节点！[核心要求简述]，你家[产品/检测]能[达标/过关]吗？#行业标签 #检测 #检测仪器 #仪器仪表
```

**特征**：
- 开头必有#话题标签作为视觉锚点
- 「新标落地」「新规出台」等时间节点制造紧迫感
- 「你家」「你能」等第二人称直击客户
- 反问句收尾，引导客户自检
- 尾部标签固定组合：行业+ #检测 #检测仪器 #仪器仪表

**示例**：
> #固态电池 新标7月落地！强势把控性能规范，你家产品的理化性能能达标吗？#新能源汽车 #检测 #检测仪器 #仪器仪表

---

### 类型2：悬念揭秘型（互动数据好）

**公式**：话题/行业 + 「受阻/卡在」+「原因竟是」+ 反转落检测

**模板**：
```
#行业 #出海/生产/合规 受阻？原因竟是[检测相关反转点]！#行业标签 #检测
```

**特征**：
- 短平快，三秒制造反转
- 「受阻+原因竟是」= 固定的悬念钩子
- 反转点必须落到检测/质检/性能验证
- 标签较少，节奏快

**示例**：
> #高分子材料 #出海 受阻？原因竟是性能不过关！#新材料 #高分子 #检测

---

### 类型3：行业趋势+行动号召型（建立专业人设）

**公式**：材料/行业定位 → 趋势判断（卷品质/标准升级） → 行动号召（质检怎么做）+ 标签收尾

**模板**：
```
#材料/行业名 作为[核心应用领域]的关键材料，[标准/性能]都在升级，[趋势判断]成了大趋势！[行动号召]？#标签1 #标签2 #标签3
```

**特征**：
- 三段式结构：定位→趋势→落地
- 适合需要铺垫行业认知的选题
- 体现专业判断力
- 尾部用反问句引导行动

**示例**：
> #先进陶瓷 作为#半导体 、#新能源 、高端装备的核心材料，安全与性能标准都在升级，卷品质成了大趋势！完善质检，紧跟风口，今天就来看看质检具体怎么做？#质检 #实验室设备

---

### 类型4：定向喊话型（精准筛选客户）

**公式**：行业/产品 + 潮/风口 + 直接@目标人群（各位XX老板）+ 反问 + 落到检测价值

**模板**：
```
#行业/产品 潮/风口即将来袭！各位#供应链环节 上的[称呼，如老板/负责人]能[做到什么]吗？不说大话，先让#检测 替你把把关#核心关键词
```

**特征**：
- 开头直接@目标人群
- 「老板」「负责人」拉近距离
- 「不说大话」制造实在、不画饼的人设感
- 「先让检测替你把把关」= 固定收尾句式
- 反问后迅速落检测价值

**示例**：
> #人形机器人 量产潮即将来袭！各位#peek #供应链 上的老板能拿出合格的材料应对生产狂潮吗？不说大话，先让#检测 替你把把关#聚醚醚酮

## 使用方式

用户发一段已定稿的脚本内容，按以下步骤处理：

### 第一步：分析脚本核心要素

从脚本中提取：
- **行业/材料**：如固态电池、高分子材料、先进陶瓷
- **检测项目**：如理化性能、重金属检测、材料性能验证
- **核心痛点或事件**：如新标落地、出海受阻、量产潮
- **时间节点（如有）**：如7月落地、年底截止

### 第二步：匹配适用类型

根据脚本特征，从四种类型中选 **1-2 个最合适的**（默认出2个版本，用户指定则出一个）：

| 脚本特征 | 推荐类型 | 原因 |
|---------|---------|------|
| 有新标/新规/政策节点 | 类型1 痛点反问 | 天然适合紧迫感+反问 |
| 有行业困境/反转点 | 类型2 悬念揭秘 | 短平快，互动好 |
| 需要铺垫行业认知 | 类型3 趋势+号召 | 建立专业人设 |
| 有明确目标人群 | 类型4 定向喊话 | 精准筛选 |
| 脚本较长、知识点多 | 类型3 + 类型1 | 先建立认知再反问收窄 |

### 第三步：生成文案

按选定的类型模板生成文案，遵守以下规则：

**通用规则：**
- 单条文案控制在 2-4 行，不超过抖音评论区可视范围
- #标签使用行业内通用标签（非自创），尾部固定标签：行业标签 + #检测 #检测仪器 #仪器仪表
- 标签数量控制在 5-8 个，避免堆砌
- 语气：直接、有料、不油腻，禁止「家人们」「绝绝子」等过度营销词
- 禁止破折号
- 禁止「不是……而是……」议论文句式

## 输出格式

```
📱 发布文案

脚本关键词：[行业] + [检测项目] + [事件/痛点]

【版本1 — [类型名]】
[文案]

【版本2 — [类型名]】
[文案]

💡 选用建议：[适用场景/平台差异/发布时机]
```
