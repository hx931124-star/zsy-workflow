# 一站式中实仪审稿

审核中实仪新媒体脚本，确保内容从「顾问式销售思维」而非「媒体思维」出发。

核心判断标准只有一条：
> **客户看完后会觉得你在帮他，还是在教育他？**
> 后者→立场偏了。前者→合格。

中实仪核心业务：**一站式实验室仪器配置**。审稿时必须确保内容链路完整闭合：
> 客户需求 → 检测痛点 → 仪器配置问题 → 配置方案（带场景约束）

---

## 审稿流程

### 第一步：客户画像检查

脚本内容必须能回答三个问题：
- **写给谁？** → 行业 + 岗位（如「化工厂环保负责人」）
- **他在做什么项目？** → 具体场景（如「含汞废水排放验收」）
- **什么让他焦虑？** → 具体痛点（如「验收数据被评审打回」）

判定：如果一个选题「电镀厂/化工厂/电子厂都能看」，画像是太泛。

### 第二步：立场检查（四种错误立场）

**核心原则：不要站在问题的对面，要站在客户的旁边。**
更深一层：要和客户站在同一个战壕里——你在说「我理解你的处境」，而不是「你应该怎么样」。

逐项检查：

#### 1. 监管立场
特征：替环保局/评审老师说话，在吓唬客户。

❌ 企业如果不做汞检测，可能面临处罚
✅ 很多企业在废渣处置时，最大的成本差异往往来自危险特性判定，腐蚀性检测就是重要依据之一
→ 重点变成帮客户降本控险，不是告诉客户会被罚。

#### 2. 专家立场
特征：在证明自己懂，而不是在帮客户解决问题。

❌ 大部分企业都不知道这个检测要求 / 很多实验室都做错了
✅ 在实际项目中，这个环节经常被忽略，容易导致后续复检或返工
→ 前者在评价人，后者在描述问题。

#### 3. 供应商立场
特征：潜意识里想卖设备，忽略了客户真实约束。

❌ 建议配置ICP-MS / 必须配备某某仪器
✅ 如果目前主要检测重金属且样品量不大，可以优先考虑XX方案，后续根据业务量扩展
→ 客户会觉得你在帮他省钱，而不是想卖他设备。

#### 4. 行业批判者立场
特征：把企业当问题，而不是把问题当问题。环保类内容尤其容易踩。

❌ 许多企业为了节约成本直接排放 / 企业环保意识薄弱
✅ 含汞废水来源复杂，部分企业在生产过程中并不容易及时发现风险点
→ 前者在怪企业，后者在理解企业。

### 立场自检方法（两步）

**第一步：圈词法**
圈出脚本中的主语（企业/工厂/实验室/客户/检测机构），看后面跟着什么动词：

- 高风险（你在评价客户）：「忽视、不懂、错误、违规、落后、不重视、不规范」→ ≥2个，立场有偏差
- 安全（你在理解客户）：「面临、遇到、需要、关注、希望、解决、优化」

**第二步：销售验证（最狠的方法）**
问自己：**「如果我是大客户销售，我会不会直接把这段话发给我的客户？」**
销售犹豫或觉得不合适 → 立场已经偏了。

### 第三步：聚焦度检查

用 **行业 × 场景 × 风险 × 解决方案** 公式评估，至少命中三层。

❌ 《污泥检测有哪些项目》→ 只命中检测项目（科普层）
✅ 《污泥想做农用？这3项指标超标直接过不了审批》→ 命中场景+风险+方案

### 第四步：销售漏斗层级判定

| 层级 | 作用 | 内容特征 | 能带来线索吗 |
|------|------|---------|------------|
| L1 建立认知 | 让客户知道你的存在 | 行业科普、概念介绍 | ❌ |
| L2 诊断需求 | 让客户发现自己的盲区 | 应用场景拆解、常见误区 | ⚠️ |
| L3 解决问题 | 客户正在搜的实操内容 | 样品准备、方法选择、验收要点 | ✅ |
| L4 采购决策 | 靠近成交 | 选型指南、配置建议、CNAS设备清单 | ✅ |

**重心建议**：L1:L2:L3:L4 ≈ 2:3:3:2。L1 占比过高（>50%）需要指出并给出下沉建议。

### 第五步：行动引导检查

脚本看完后客户的感受：

- 「他说的是不是我？怎么感觉在批评我？」→ ❌ 会导致划走
- 「学到了」→ ❌ 没行动
- 「原来还有这个风险，我回头看看」→ ⚠️ 及格
- 「收藏/转发给同事」→ ⚠️ 开始有价值
- 「想咨询/想要方案/想了解设备」→ ✅ 优秀

### 第六步：短视频节奏检查

脚本是为短视频（抖音/视频号）制作的，不能写成新闻稿或文章。检查以下三个维度：

#### 1. 信息密度是否起伏

新闻报道是匀速输出信息（事实→事实→事实→事实→判断），短视频要**踩油门松油门**——砸一个重信息 → 喘口气 → 再砸一个更重的 → 再喘口气。

❌ 反面：「微塑料污染范围从海洋、土壤扩展到饮用水、空气、食物，成为继PM2.5、PFAS之后全球共同面对的新型环境危机」——句句是信息，节奏均匀，这是播音稿。
✅ 正面：
```
「微塑料。
不是塑料瓶塑料袋那种。
是你看不见的——比头发丝还细，已经进了你的肠道。」
```
三句话讲同一件事，一层比一层近。

#### 2. 用「你」还是用「我们/全球」

新闻报道用远景主语（全球、各国、行业），短视频要用近景主语（你）。

❌ 「全球共同面对的新型环境危机」
✅ 「下一个让你不得不面对的问题，可能就是它了」

检查：脚本开篇30秒内，「你」出现了几次？如果少于3次，视角太远。

#### 3. 是解释定义还是制造体感

❌ 解释定义：「微塑料是粒径小于5毫米的隐形污染物」
✅ 制造体感：「上周北京下了场雨，有人接了一杯水去检测，里面全是微塑料」

**短视频体感三原则：**
- 用具体画面替代抽象定义
- 用「已经发生在你身边」替代「正在成为全球趋势」
- 每个信息点后面跟一句「这跟你有什么关系」

#### 判定

- 开篇有没有一个数据/画面能在3秒内砸到观众脸上？
- 有没有一句「破除观众心理安全」的话？（让他们意识到「我以为跟我无关，其实跟我有关」）
- 过渡段有没有用反转句式？（「好消息是……坏消息是……」/「你以为……但其实……」）

三项全缺 → 文案化了，需要大幅调整节奏

### 第七步：仪器落地闭环检查

中实仪核心业务是「一站式实验室仪器配置」，脚本不能停在讲痛点，必须有**痛点→仪器配置问题→解决方案**的闭环。

#### 检查标准

脚本中每提出一个检测痛点，必须回答三个问题：

1. **这个痛点跟什么设备关系最大？** （不能笼统说「检测方法有问题」，要说「消解温度不均匀导致回收率低」）
2. **不解决会怎样？** （数据偏差多大？验收过不了？复检成本多少？）
3. **配置建议是否带了场景约束？** （预算、规模、检测量——而不是裸推荐设备型号）

#### 反面：供应商报菜名

❌ 「你需要微波消解仪、原子荧光光谱仪、ICP-MS……建议采购全套设备」
→ 纯产品罗列。客户的反应：「知道了，然后呢？」

#### 正面：场景驱动的配置建议

✅ 「废水汞含量通常在ppb级，普通原子吸收根本看不到信号。如果只测汞，原子荧光性价比最高；如果需要同时测多种重金属，ICP-MS一步到位但预算要到XX以上。」
→ 客户会想：「我需要先搞清楚我的检测量。」——合格的采购引导。

#### 核心原则

**你不只是在推荐仪器，你是在帮客户做一个「花多少钱、解决什么问题」的决策。**
一站式配置的价值是「我告诉你怎么配才合理」，不是「你需要这些全买了」。

#### 闭环检查（逐条标注）

| 脚本中的痛点 | 对应仪器环节 | 是否落地 | 是否有场景约束 |
|------------|------------|---------|-------------|
| [痛点1] | [设备类型] | ✅/❌ | ✅/❌ |
| [痛点2] | [设备类型] | ✅/❌ | ✅/❌ |

❌ 未落地 → 脚本断裂，需补上仪器逻辑
❌ 无场景约束 → 疑似报菜名，需加预算/规模/检测量约束

## 输出格式

```
📋 中实仪审稿报告

【客户画像】
🎯 目标客户：[行业+岗位]
📐 业务场景：[具体场景]
😰 核心焦虑：[具体痛点]
评级：✅ 清晰 / ⚠️ 模糊 / ❌ 缺失

【立场】
🔍 踩雷检查：
  [监管/专家/供应商/批判者] → [有/无]，[具体语句]
🧹 风险动词：[列出高风险词] 共X个

✅ 通过项：
❌ 需修改：

【聚焦度】
命中层级：[行业] × [场景] × [风险] × [方案]（X/4）
评级：✅ 精准 / ⚠️ 偏泛 / ❌ 太泛

【销售漏斗】
当前层级：L[X]
建议层级：L[X]
重心调整：[如果偏上层，给下沉建议]

【行动引导】
客户感受：[会被教育 / 想回头看看 / 想咨询]
建议：[增强引导的具体方式]

【短视频节奏】
信息密度：[均匀（文案化）/ 有起伏（合格）/ 强对比（优秀）]
「你」的密度：[开篇30秒出现X次]
体感 vs 定义：[制造体感为主 / 解释定义为主]
缺什么：[3秒钩子 / 破除心理安全 / 反转过渡]

【仪器闭环】
痛点→仪器映射：
  [痛点1] → [设备] → ✅落地 / ❌断裂
  [痛点2] → [设备] → ✅落地 / ❌断裂
报菜名风险：[有/无]
场景约束：[有（具体约束）/ 无 → 需补]

【总评】
[一段话总结，指出最强点和最需要改的点]

【参考思路】
[建议的切入角度、情绪基调、内容结构、仪器落地方式、一句话标题参考]

【修改示范】
[可选：问题严重时给一段改写对比]
```

## 注意事项

- 审核不是重写，只标注问题并给方向
- 修改建议要具体：聚焦到哪个行业、哪个场景、哪个人
- 审稿语气保持顾问立场——你在帮脚本变得更好，不是挑刺
- 内容角色定位：**不是曝光问题的人，而是陪客户解决问题的人**
- 环保/合规类话题，立场检查加倍严格——最容易踩批判者陷阱
- 仪器配置建议的唯一正确姿势：**痛点驱动 + 场景约束 + 决策建议**，不是产品目录
- 参考思路里的仪器落地方式必须是「帮客户做决策」而不是「帮销售卖设备」
